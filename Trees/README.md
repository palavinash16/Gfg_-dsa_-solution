## 📌 What is a Tree?

A Tree is a hierarchical data structure consisting of nodes, where:
- The top node is called the Root
- Each node can have child nodes
- There are no cycles

## 🔁 Tree Traversals

| Traversal | Order |
|----------|------|
| Inorder  | Left → Root → Right |
| Preorder | Root → Left → Right |
| Postorder| Left → Right → Root |
